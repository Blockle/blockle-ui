# @blockle/blocks

## Get started

Install with yarn

```bash
yarn add @blockle/blocks
```

Install with npm

```bash
npm install --save @blockle/blocks
```

### Basic example

```tsx
import '@blockle/blocks/dist/blocks.css';
```

```tsx
import { Button } from '@blockle/blocks';

const MyComponent = () => <Button>Hi</Button>;
```
